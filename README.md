# file_server.rs

Simple file server and blogging engine

## Routes

- GET /login
- *GET /logout*
- *GET /blog/&lt;title&gt;*
- *POST /blog/*
- *GET /new*
- GET /static/&lt;file&gt;

  *highlighted routes need the user to be logged in*
